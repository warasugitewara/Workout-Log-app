# Workout Log Pro

筋トレ管理Androidアプリ

## 技術スタック
- **言語**: Kotlin
- **UI**: Jetpack Compose + Material3
- **DB**: Room (SQLite)
- **通知**: WorkManager
- **グラフ**: MPAndroidChart
- **アーキテクチャ**: MVVM
- **対象OS**: Android 10+

## ビルド
```
./gradlew assembleDebug
```

## 機能一覧

### Phase 1 - MVP ✅
- トレーニングメニュー管理 (CRUD)
- トレーニングログ記録 (回数・セット・時間・カロリー・体脂肪率・疲労度)
- 体重推移グラフ
- リマインド通知 (WorkManager / 曜日別時刻指定)
- ダッシュボード (統計サマリー)
- ユーザー情報管理 (BMI / 基礎代謝自動計算)

### Phase 2 - 統計拡張 ✅
- BMI推移グラフ
- 種目別回数推移グラフ
- 週間・月間消費カロリーグラフ
- トレーニング実施率 (週/月)
- 平均タイム統計
- 21種メニューテンプレート (上半身/下半身/有酸素/体幹)
- 12種ルーティンテンプレート (筋肥大/有酸素/体幹/初心者/HIIT/PPL)
- 週間スケジュール管理画面
- スケジュール完了時の自動ログ記録
- カロリー推定表示
- セットトラッキングモード (まとめて/1セット毎)
- プロテイン記録管理 (摂取量/割り方/日計)

### Phase 3 - データ管理 ✅
- CSVエクスポート (トレーニングログ/プロテインログ)
- ダークテーマ対応 (システム設定連動)
- Dynamic Color対応 (Android 12+)
