# Workout Log Pro 🏋️‍♂️

筋トレ管理Androidアプリ。モダンなUIと詳細な統計機能で、あなたのワークアウトを強力にサポートします。

## はじめに
今回は、筋トレ管理アプリを制作しました。私の好みのUI,UXで、色々と最適化の余地やバグはありますが、随時更新していきます。

## 🌟 特徴
- **GitHubスタイルの洗練されたデザイン**: スタイリッシュなダークモード（High Contrast）と清潔感のあるライトモード。
- **エアロバイク・アイコン**: フィットネスへの情熱を象徴するカスタムアプリアイコン。
- **直感的なダッシュボード**: 今日の進捗、体重、BMI、消費カロリーを一目で把握。
- **詳細なセット管理**: 「まとめて記録」または「1セット毎にチェック」を選択可能。

## 🛠 技術スタック
- **言語**: Kotlin
- **UI**: Jetpack Compose + Material3 (GitHub Theme)
- **DB**: Room (SQLite)
- **通知**: WorkManager
- **グラフ**: MPAndroidChart
- **アーキテクチャ**: MVVM
- **対象OS**: Android 10+ (API 29+)

## 🚀 ビルド & 実行
```bash
# APKの生成
./gradlew assembleDebug

# デバイスへのインストール
./gradlew installDebug
```

## 機能一覧

### Phase 1 - MVP ✅
- トレーニングメニュー管理 (CRUD)
- トレーニングログ記録 (回数・セット・時間・カロリー・体脂肪率・疲労度)
- 体重推移グラフ
- リマインド通知 (WorkManager / 曜日別時刻指定)
- ダッシュボード (統計サマリー & クイックアクション)
- ユーザー情報管理 (BMI / 基礎代謝自動計算)
- **カスタム・エアロバイクアイコン** 🚲

### Phase 2 - 統計 & スケジュール ✅
- BMI推移グラフ
- 種目別回数推移グラフ
- 週間・月間消費カロリーグラフ
- トレーニング実施率 (週/月)
- 平均タイム統計
- 21種メニューテンプレート (上半身/下半身/有酸素/体幹)
- 12種ルーティンテンプレート (筋肥大/有酸素/体幹/初心者/HIIT/PPL)
- 週間スケジュール管理画面
- スケジュール完了時の自動ログ記録
- **セットトラッキングモードの完全同期** (1セット毎の分割管理に対応)
- プロテイン記録管理 (摂取量/割り方/日計)

### Phase 3 - UI/UX 刷新 & データ管理 ✅
- **GitHub Inspired Theme**: スタイリッシュな配色とタイポグラフィ。
- CSVエクスポート (トレーニングログ/プロテインログ)
- モダンなアニメーションとカードデザイン
- パフォーマンス最適化
